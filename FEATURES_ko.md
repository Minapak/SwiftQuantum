# SwiftQuantum iOS 앱 - 완전한 기능 문서

**버전**: 2.2.4
**플랫폼**: iOS 18+ / macOS 15+
**언어**: 영어, 한국어, 일본어, 중국어 (간체), 독일어

---

## 개요

SwiftQuantum은 로컬 시뮬레이션, 실제 IBM Quantum 하드웨어 연결, 교육 콘텐츠를 결합한 iOS용 프리미엄 양자 컴퓨팅 플랫폼입니다. 앱은 Apple Human Interface Guidelines에 기반한 4-Hub 네비게이션 시스템을 사용합니다.

### 네비게이션 구조

| Hub | 탭 이름 | 아이콘 | 설명 |
|-----|-------|------|------|
| 1 | Lab | `atom` | 양자 상태 제어 및 측정 |
| 2 | Circuits | `square.grid.3x3` | 사전 구축된 회로 템플릿 및 빌더 |
| 3 | Factory | `cpu.fill` | QuantumBridge QPU 연결 |
| 4 | More | `ellipsis` | Academy, Industry, 프로필, 설정 |

---

## 탭 1: Lab Hub

Lab Hub는 컨트롤, 측정, 정보 패널을 결합한 핵심 양자 실험 인터페이스입니다.

### 1.1 컨트롤 패널

**목적**: 확률 및 위상 컨트롤을 통해 양자 상태를 조작합니다.

| 기능 | 설명 | 상호작용 |
|-----|------|---------|
| 확률 슬라이더 | |0⟩ 및 |1⟩ 확률 진폭 조정 | 슬라이더 0-100% 드래그 |
| 위상 슬라이더 | 상태 간의 상대적 위상 제어 | 슬라이더 0-2π 드래그 |
| 블로흐 구 | 큐빗 상태의 3D 시각화 | 회전, 줌 제스처 |
| 상태 벡터 표시 | α|0⟩ + β|1⟩ 표기법 표시 | 읽기 전용 |

**사용 가능한 양자 게이트**:

| 게이트 | 기호 | 행렬 | 효과 |
|-------|-----|------|------|
| Hadamard | H | 1/√2 [[1,1],[1,-1]] | 중첩 생성 |
| Pauli-X | X | [[0,1],[1,0]] | 비트 플립 (NOT) |
| Pauli-Y | Y | [[0,-i],[i,0]] | 비트 + 위상 플립 |
| Pauli-Z | Z | [[1,0],[0,-1]] | 위상 플립 |
| S 게이트 | S | [[1,0],[0,i]] | π/2 위상 |
| T 게이트 | T | [[1,0],[0,e^(iπ/4)]] | π/8 위상 |

### 1.2 측정 패널

**목적**: 양자 측정을 수행하고 통계적 결과를 봅니다.

| 기능 | 설명 | 출력 |
|-----|------|------|
| 단일 샷 | 상태를 |0⟩ 또는 |1⟩로 붕괴 | 단일 결과 |
| 다중 샷 | 여러 측정 실행 | 히스토그램 |
| 샷 수 | 10-10000 샷 구성 | 슬라이더 |
| 결과 기록 | 과거 측정 보기 | 스크롤 가능한 목록 |

### 1.3 정보 패널

**목적**: 현재 양자 상태에 대한 교육 콘텐츠.

| 섹션 | 내용 |
|-----|------|
| 상태 설명 | 현재 상태에 대한 평이한 언어 설명 |
| 수학적 표기법 | 디랙 표기법, 밀도 행렬 |
| 블로흐 좌표 | 블로흐 구의 θ, φ 각도 |
| 역사적 맥락 | 관련 양자 물리학 개념 |

## 탭 2: Circuits Hub (프리셋)

Circuits Hub는 사전 구축된 양자 회로 템플릿과 회로 빌더를 제공합니다.

### 2.1 빠른 시작 섹션

**목적**: 일반적인 양자 회로에 대한 원탭 접근.

| 회로 | 큐빗 | 설명 |
|-----|------|------|
| Bell 상태 | 2 | 최대 얽힌 쌍 생성 |
| GHZ 상태 | 3 | 3큐빗 얽힘 |
| 중첩 | 1 | |+⟩ 상태의 단일 큐빗 |
| 비트 플립 | 1 | X 게이트 시연 |

### 2.2 주요 회로

**목적**: 고급 양자 알고리즘 소개.

| 카테고리 | 회로 |
|---------|------|
| **기초** | 중첩, 위상 킥백, 간섭 |
| **얽힘** | Bell 상태, GHZ, W 상태, CHSH |
| **알고리즘** | Deutsch-Jozsa, Bernstein-Vazirani, Simon |
| **고급** | Grover 검색, QFT, VQE, QAOA |

### 2.3 회로 템플릿

**전체 템플릿 라이브러리**:

| 템플릿 | 큐빗 | 게이트 | 난이도 | 프리미엄 |
|-------|------|-------|--------|---------|
| 기본 중첩 | 1 | 1 | 초급 | 무료 |
| Bell 상태 | 2 | 2 | 초급 | 무료 |
| GHZ 상태 | 3 | 3 | 초급 | 무료 |
| 위상 추정 | 4 | 12 | 중급 | Pro |
| Grover 알고리즘 | 3 | 15 | 중급 | Pro |
| 양자 푸리에 변환 | 4 | 20 | 고급 | Premium |
| Shor 알고리즘 (데모) | 5 | 50+ | 고급 | Premium |

## 탭 3: Factory Hub (Bridge)

Factory Hub는 QuantumBridge를 통해 실제 IBM Quantum 컴퓨터에 연결합니다.

### 3.1 연결 상태

| 상태 | 아이콘 | 설명 |
|-----|-------|------|
| 연결됨 | 녹색 점 | 백엔드에 활성 연결 |
| 연결 끊김 | 회색 점 | 활성 연결 없음 |
| 연결 중 | 스피너 | 연결 설정 중 |
| 오류 | 빨간 점 | 연결 실패 |

### 3.2 백엔드 선택

**사용 가능한 백엔드**:

| 백엔드 | 큐빗 | 최적 용도 | 대기열 시간 |
|-------|------|---------|-----------|
| **로컬 시뮬레이터** | 20 | 테스트 | 즉시 |
| **IBM Brisbane** | 127 | 프로덕션 | 5-30분 |
| **IBM Osaka** | 127 | 빠른 실험 | 2-15분 |
| **IBM Kyoto** | 127 | 연구 | 10-60분 |

### 3.3 빠른 작업

| 작업 | 설명 | 프리미엄 |
|-----|------|---------|
| **Bell 상태** | 2큐빗 얽힘 생성 | Pro |
| **GHZ 상태** | 3큐빗 얽힘 생성 | Pro |
| **QASM 내보내기** | OpenQASM 3.0으로 회로 복사 | 무료 |
| **연속 모드** | 30초마다 자동 반복 | Premium |

### 3.4 QASM 내보내기 시트

**내보내기 옵션**:
- 클립보드에 복사
- 시스템 공유 시트를 통해 공유
- 구문 강조 코드 보기

**QASM 3.0 예제**:
```qasm
OPENQASM 3.0;
include "stdgates.inc";

qubit[2] q;
bit[2] c;

reset q;
h q[0];
cx q[0], q[1];
c = measure q;
```

### 3.5 오류 정정 패널 (Premium)

| 기능 | 설명 |
|-----|------|
| 표면 코드 시각화 | 데이터 및 보조 큐빗의 시각적 그리드 |
| 코드 거리 | 구성 가능한 d=3, d=5, d=7 |
| 논리적 오류율 | 물리적 오류율을 기반으로 계산 |
| 충실도 표시 | 정정 후 측정 충실도 |
