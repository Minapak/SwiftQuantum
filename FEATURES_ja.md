# SwiftQuantum iOSアプリ - 完全な機能ドキュメント

**バージョン**: 2.2.4
**プラットフォーム**: iOS 18+ / macOS 15+
**言語**: 英語、韓国語、日本語、中国語（簡体字）、ドイツ語

---

## 概要

SwiftQuantumは、ローカルシミュレーション、実際のIBM Quantumハードウェア接続、教育コンテンツを組み合わせたiOS向けプレミアム量子コンピューティングプラットフォームです。アプリはApple Human Interface Guidelinesに基づいた4-Hubナビゲーションシステムを使用しています。

### ナビゲーション構造

| Hub | タブ名 | アイコン | 説明 |
|-----|-------|--------|------|
| 1 | Lab | `atom` | 量子状態の制御と測定 |
| 2 | Circuits | `square.grid.3x3` | プリビルト回路テンプレートとビルダー |
| 3 | Factory | `cpu.fill` | QuantumBridge QPU接続 |
| 4 | More | `ellipsis` | Academy、Industry、プロファイル、設定 |

---

## タブ1: Lab Hub

Lab Hubは、コントロール、測定、情報パネルを組み合わせたコア量子実験インターフェースです。

### 1.1 コントロールパネル

**目的**: 確率と位相のコントロールを通じて量子状態を操作します。

| 機能 | 説明 | インタラクション |
|-----|------|----------------|
| 確率スライダー | |0⟩と|1⟩の確率振幅を調整 | スライダー0-100%をドラッグ |
| 位相スライダー | 状態間の相対位相を制御 | スライダー0-2πをドラッグ |
| ブロッホ球 | キュービット状態の3D可視化 | 回転、ズームジェスチャー |
| 状態ベクトル表示 | α|0⟩ + β|1⟩表記を表示 | 読み取り専用 |

**使用可能な量子ゲート**:

| ゲート | 記号 | 行列 | 効果 |
|-------|-----|------|------|
| Hadamard | H | 1/√2 [[1,1],[1,-1]] | 重ね合わせを作成 |
| Pauli-X | X | [[0,1],[1,0]] | ビットフリップ（NOT） |
| Pauli-Y | Y | [[0,-i],[i,0]] | ビット＋位相フリップ |
| Pauli-Z | Z | [[1,0],[0,-1]] | 位相フリップ |
| Sゲート | S | [[1,0],[0,i]] | π/2位相 |
| Tゲート | T | [[1,0],[0,e^(iπ/4)]] | π/8位相 |

### 1.2 測定パネル

**目的**: 量子測定を実行し、統計的結果を表示します。

| 機能 | 説明 | 出力 |
|-----|------|------|
| シングルショット | 状態を|0⟩または|1⟩に崩壊 | 単一結果 |
| マルチショット | 複数の測定を実行 | ヒストグラム |
| ショット数 | 10-10000ショットを設定 | スライダー |
| 結果履歴 | 過去の測定を表示 | スクロール可能なリスト |

| Bell状態 | 2 | 2 | 初級 | 無料 |
| GHZ状態 | 3 | 3 | 初級 | 無料 |
| 位相推定 | 4 | 12 | 中級 | Pro |

## タブ3: Factory Hub（Bridge）

Factory HubはQuantumBridgeを通じて実際のIBM Quantumコンピューターに接続します。

### 3.1 接続状態

| 状態 | アイコン | 説明 |
|-----|--------|------|
| 接続中 | 緑のドット | バックエンドへのアクティブな接続 |
| 切断 | グレーのドット | アクティブな接続なし |
| 接続中 | スピナー | 接続を確立中 |
| エラー | 赤のドット | 接続失敗 |

### 3.2 バックエンド選択

**利用可能なバックエンド**:

| バックエンド | キュービット | 最適用途 | キュー時間 |
|------------|------------|---------|----------|
| **ローカルシミュレータ** | 20 | テスト | 即時 |
| **IBM Brisbane** | 127 | 本番 | 5-30分 |
| **IBM Osaka** | 127 | 高速実験 | 2-15分 |
| **IBM Kyoto** | 127 | 研究 | 10-60分 |

### 3.3 クイックアクション

| アクション | 説明 | プレミアム |
|----------|------|----------|
| **Bell状態** | 2キュービットエンタングルメント作成 | Pro |
| **GHZ状態** | 3キュービットエンタングルメント作成 | Pro |
| **QASMエクスポート** | OpenQASM 3.0として回路をコピー | 無料 |
| **連続モード** | 30秒ごとに自動繰り返し | Premium |
