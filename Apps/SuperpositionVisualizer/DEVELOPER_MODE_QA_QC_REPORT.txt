PART 5: HOW TO USE DEVELOPER MODE
================================================================================

5.1 Enabling Developer Mode
---------------------------
Developer Mode is enabled by default in DeveloperModeManager.swift:
@Published var isEnabled: Bool = true

To disable, change to:
@Published var isEnabled: Bool = false

5.2 Viewing Tap Logs
--------------------
1. Launch the app
2. Tap the red "DEV MODE [N]" badge in the top-left corner
3. Full-screen overlay shows:
   - Total interactions count
   - Success/Failed/Coming Soon/No Action statistics
   - Chronological list of all tapped elements with timestamps

5.3 Clearing Logs
-----------------
Tap the "Clear" button in the log overlay to reset all logs.

5.4 Log Format
--------------
[HH:mm:ss.SSS] [Status Emoji] [Screen] Element Name

Example:
[14:32:15.234] ✅ [Lab] Gate: Hadamard
[14:32:18.567] ⏳ [Bridge] Backend: IBM Brisbane (Premium)
